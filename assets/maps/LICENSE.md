# World map asset license

`world_equal_earth.svg` is a derived, simplified landmass outline generated from Natural Earth 1:110m country polygons. Natural Earth data is in the public domain. The derived SVG is distributed under the same public-domain terms.

The map is intended as a neutral analytical base layer. It does not encode or assert political boundaries, sovereignty, or territorial claims. Jurisdiction labels and classifications are supplied separately by the deck project and must be supported by project evidence.
