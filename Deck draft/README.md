# Deck draft

Use this directory for repo-backed process state created during real deck generation.

Recommended project path:

```text
Deck draft/<YYYY-MM-DD>/<deck-title-slug>/
```

Each project should contain `storyline.md`, `briefs.yaml`, `evidence.json`, `assumptions.md`, `pages/`, `output/`, `baseline/`, and `changelog.md` as defined in `references/project-state.md`.

Do not keep intermediate process artifacts only in local scratch files or chat history.
